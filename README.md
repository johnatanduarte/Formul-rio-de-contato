# Formulário de Orçamento

Este projeto é uma solução de um desafio de Front-end que consiste em reproduzir fielmente um design de formulário de contato. O foco principal foi a implementação do efeito "Floating Label" (onde o rótulo sobe ao clicar no campo) e a validação de formulário via JavaScript puro.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica.
* **CSS:** Estilização, Flexbox, efeitos de transição e responsividade.
* **JavaScript:** Lógica de validação de campos e manipulação do DOM.

## Funcionalidades

* **Floating Labels:** O placeholder se transforma em label e "flutua" para o topo ao focar no input.
* **Validação em Tempo Real:** Verificação de campos vazios e formato de e-mail sem recarregar a página.
* **Feedback Visual:** Bordas vermelhas para erros e mensagem de sucesso ao enviar.
* **Design Responsivo:** Adaptação para dispositivos móveis e desktops.
* **Custom Select:** Estilização personalizada da seta do campo de seleção (`<select>`).

## Como rodar o projeto localmente

Uma das formas de visualizar este projeto na sua máquina é a seguinte: você precisará do **VS Code** e de uma extensão chamada **Live Server**.

### Passo a Passo

1.  **Clone o repositório** (ou baixe o ZIP):
    ```bash
     git remote add origin https://github.com/johnatanduarte/Formul-rio-de-contato.git
    ```

2.  **Abra a pasta no VS Code**:
    Abra o VS Code, vá em `File` > `Open Folder` e selecione a pasta do projeto.

3.  **Instale a extensão recomendada**:
    * No menu lateral do VS Code, clique no ícone de **Extensões**.
    * Pesquise por **"Live Server"**.
    * Clique em **Install**.

4.  **Rode o projeto**:
    * Abra o arquivo `index.html`.
    * Clique com o **botão direito** em qualquer lugar do código.
    * Selecione a opção **"Open with Live Server"**.
    * O navegador abrirá automaticamente com o projeto rodando.
